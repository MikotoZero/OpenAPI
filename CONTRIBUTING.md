## Bug 上报

可以使用 GitHub 的 [issue tracker][issue] 来上报发现的问题：

[issue]: https://github.com/bearyinnovative/OpenAPI/issues/new

## 开发社区

可以加入 [BearyChat OpenAPI][openapi] 团队来提问 & 回答关于 OpenAPI 的问题。

[openapi]: https://openapi.bearychat.com/apply

## 本地开发

1. fork repo
2. 进入对应修改的目录
3. 修改对应文件
4. npm run build
5. commit
6. 发起 pull request
7. **benefit!!!1**
